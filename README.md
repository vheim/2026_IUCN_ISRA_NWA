# 2026_IUCN_ISRA_NWA
R code for analyses completed to support marine waters between the GOM and the U.S. East coast being listed as a C4-criteria based Important Shark And Ray Area (ISRA) for species of the large-bodied hammerhead complex, i.e. great, scalloped, and smooth hammerheads (Sphyrna mokarran, S. lewini, and S. zygaena) in the western North Atlantic.
